# TF Specs

Specs are now dynamically generated based on your connected GitHub repository.

See the main `/specs/INDEX.md` for details on how the new spec system works.

## Quick Start

1. Start the Testing Framework (`Start-TF.bat`)
2. Open http://localhost:3001
3. Connect your GitHub repository (token + owner/repo)
4. Click "Generate Specs" to create specs for your app
5. Select a spec and start testing
